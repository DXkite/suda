# 最终PHP坑：苏打 简易PHP框架
此框架的目的是简化PHP程序的构建过程，不包含太多东西。是我尝试使用面向对象的思路去编写的`DxCore`,
老式的处理思路导致我在面向对象设计的过程中还保留很多过去面向函数编程的习惯。因此，本程序尝试使用新的思路
去编写代码，而且，如标题所言，这个是我最后的PHP自用辅助类库。

## 框架功能
- 单入口
- MVC
- 模块化


## 主要辅助功能：
- 控制台自动构建功能
- 应用路由
- 事件监听器
- 简易PHP模板
- 日志记录工具
